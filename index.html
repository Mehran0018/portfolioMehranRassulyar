Project: Crypto Tracker

In dit project heb ik een simpele website gemaakt waarmee je de prijs van een cryptocurrency (zoals bitcoin) kunt opzoeken. De website gebruikt een echte API (CoinGecko) om de actuele prijs op te halen en toont dit ook in een grafiek. Je ziet het verschil in prijs per zoekopdracht.

Wat ik heb geleerd: API's gebruiken, prijsverschillen tonen, grafieken maken met Chart.js en JavaScript toepassen.
