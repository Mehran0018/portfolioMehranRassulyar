Project: Crypto Tracker

In dit project heb ik een simpele website gemaakt waarmee je de prijs van een cryptocurrency (zoals bitcoin) kunt opzoeken. De website gebruikt een echte API (CoinGecko) om de actuele prijs op te halen. Elke keer als je zoekt, wordt de prijs ook in een grafiek gezet. Zo kun je zien hoe de prijs verandert.

Wat ik heb geleerd: werken met API's, prijzen ophalen, verschil in prijs laten zien en een grafiek bijhouden met JavaScript en Chart.js.
